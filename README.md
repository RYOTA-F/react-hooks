# React Samples

## Setup

1. Clone

```sh
$ git clone https://github.com/RYOTA-F/react-samples.git
```

2. Module Install

```sh
$ yarn install
```

### Firebase Setting

1. Copy .env

```sh
$ cp .env_example .env
```

2. Edit .env

```sh
$ vi .env
```

## Run

```sh
$ yarn start
```

## Test

```sh
$ yarn test
```

## Versions

- typescript: 4.6.2
- react: 17.0.2
- react-redux: 8.0.2
- swr: 1.3.0
